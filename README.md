# School Market Sim

Multi-tenant student marketplace frontend built with Next.js, TypeScript, and TailwindCSS.

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000/

## Pages
- Landing: `/`
- Marketplace: `/marketplace`
- Product Detail: `/product/[id]`
- Store Management: `/store`
- Dashboard: `/dashboard`
- Wallet: `/wallet`
- Admin: `/admin`

## Spec
Minimal Phase 0 API spec at `spec/openapi.yaml`.
